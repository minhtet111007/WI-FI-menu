# WI-FI menu written in Bash

- This little script can run in terminal and be used to select network.
- But due to being a beginner, I've tried my best.
- The first line shebang can be changed, so it's better to `which bash` before you use
- If you can't input a password to unsaved network, this may due to nmcli's saving connection that you've entered with invalid password.
- To solve this, `nmcli connection delete ssid`.
